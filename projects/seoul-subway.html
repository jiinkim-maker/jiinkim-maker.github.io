🚇 Seoul Subway Congestion Visualization & Prediction

Public Data Visualization Project | Data Analytics [2023]

📌 Purpose

This project was inspired by a service in the "Tmap" app that predicts subway congestion by car. The goal was to utilize public transportation data to visualize congestion patterns in Seoul's subway system. The visualization highlights when and where (by line and station) congestion occurs based on boarding and alighting counts.

🧠 Technical Keywords
Public Data Data Visualization Subway Congestion Data Analysis
📝 Description
  • Gathered public data on boarding and alighting counts by subway line, station, and time slot.
  • Analyzed congestion trends based on peak and off-peak hours across the Seoul subway system.
  • Visualized congestion heatmaps to make it easier to understand the level of crowding at a glance.
  • Proposed future improvements on congestion prediction using time-series modeling.
👩‍💻 Role
  • Collected and cleaned public data regarding Seoul subway usage.
  • Created interactive visual maps to showcase congestion levels across time and stations.
🏁 Results
  • Successfully produced a congestion visualization map based on real-world public data.
  • Explored further directions for enhancing prediction accuracy using time-based models.
