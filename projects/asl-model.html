🤟 ASL Sign Language Recognition AI

Course Project | Biomedical Engineering [2024]
📌 Purpose

To support communication with the hearing-impaired using an AI model that recognizes ASL hand gestures and converts them into alphabet outputs.

🛠 Technical Keywords
ASL Recognition CNN ResNeXt50 Data Augmentation
📝 Description
  • Studied CNN-based research papers and built the model from scratch without using pre-trained backbones.
  • Analyzed misclassification issues caused by visually similar hand gestures.
  • Proposed two solutions: data augmentation (rotation, flipping) and model replacement (ResNeXt50).
  • Demonstrated improved accuracy through performance experiments.
👩‍💻 Role
  • Led data collection and annotation for ASL hand gestures.
  • Implemented and tuned ResNeXt50 architecture.
🏆 Results
  • Ranked 1st in course project evaluation.
  • Achieved Accuracy: 0.834
